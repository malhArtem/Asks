### **1. –ú–Ω–æ–≥–æ–ø—Ä–æ—Ü–µ—Å—Å–æ—Ä–Ω–æ—Å—Ç—å (Multiprocessing)**  
**–ß—Ç–æ —ç—Ç–æ:**  
–ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ **–Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö –ø—Ä–æ—Ü–µ—Å—Å–æ–≤**, –∫–æ—Ç–æ—Ä—ã–µ –≤—ã–ø–æ–ª–Ω—è—é—Ç—Å—è **–ø–∞—Ä–∞–ª–ª–µ–ª—å–Ω–æ** –Ω–∞ —Ä–∞–∑–Ω—ã—Ö —è–¥—Ä–∞—Ö CPU.  
**–û—Å–æ–±–µ–Ω–Ω–æ—Å—Ç–∏:**  
- –ö–∞–∂–¥—ã–π –ø—Ä–æ—Ü–µ—Å—Å –∏–º–µ–µ—Ç **—Å–æ–±—Å—Ç–≤–µ–Ω–Ω—É—é –ø–∞–º—è—Ç—å** (–∏–∑–æ–ª–∏—Ä–æ–≤–∞–Ω –æ—Ç –¥—Ä—É–≥–∏—Ö).  
- –ü–æ–¥—Ö–æ–¥–∏—Ç –¥–ª—è **CPU-–∏–Ω—Ç–µ–Ω—Å–∏–≤–Ω—ã—Ö –∑–∞–¥–∞—á** (–Ω–∞–ø—Ä–∏–º–µ—Ä, –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏–µ –≤—ã—á–∏—Å–ª–µ–Ω–∏—è).  
- –í—ã—Å–æ–∫–∏–µ –Ω–∞–∫–ª–∞–¥–Ω—ã–µ —Ä–∞—Å—Ö–æ–¥—ã –∏–∑-–∑–∞ —Å–æ–∑–¥–∞–Ω–∏—è –ø—Ä–æ—Ü–µ—Å—Å–æ–≤.  

**–ü—Ä–∏–º–µ—Ä –Ω–∞ Python:**  
```python
import multiprocessing

def task(number):
    print(f"–ü—Ä–æ—Ü–µ—Å—Å {number} –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è")

if __name__ == "__main__":
    processes = []

        processes.append(p)
        p.start()
    for p in processes:
        p.join()
```

### **2. –ú–Ω–æ–≥–æ–ø–æ—Ç–æ—á–Ω–æ—Å—Ç—å (Multithreading)**  
**–ß—Ç–æ —ç—Ç–æ:**  
–ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ **–Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö –ø–æ—Ç–æ–∫–æ–≤** –≤ —Ä–∞–º–∫–∞—Ö –æ–¥–Ω–æ–≥–æ –ø—Ä–æ—Ü–µ—Å—Å–∞. –ü–æ—Ç–æ–∫–∏ –¥–µ–ª—è—Ç –æ–±—â—É—é –ø–∞–º—è—Ç—å.  
**–û—Å–æ–±–µ–Ω–Ω–æ—Å—Ç–∏:**  
- –ü–æ–¥—Ö–æ–¥–∏—Ç –¥–ª—è **I/O-–∏–Ω—Ç–µ–Ω—Å–∏–≤–Ω—ã—Ö –∑–∞–¥–∞—á** (–Ω–∞–ø—Ä–∏–º–µ—Ä, —Å–µ—Ç–µ–≤—ã–µ –∑–∞–ø—Ä–æ—Å—ã, —á—Ç–µ–Ω–∏–µ —Ñ–∞–π–ª–æ–≤).  
- –í Python –∏–∑-–∑–∞ **GIL (Global Interpreter Lock)** –ø–æ—Ç–æ–∫–∏ –Ω–µ –≤—ã–ø–æ–ª–Ω—è—é—Ç—Å—è –ø–∞—Ä–∞–ª–ª–µ–ª—å–Ω–æ, –∞ –ø–µ—Ä–µ–∫–ª—é—á–∞—é—Ç—Å—è –º–µ–∂–¥—É —Å–æ–±–æ–π.  

**–ü—Ä–∏–º–µ—Ä –Ω–∞ Python:**  
```python
import threading

def task(number):
    print(f"–ü–æ—Ç–æ–∫ {number} –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è")

threads = []

    t = threading.Thread(target=task, args=(i,))
    threads.append(t)
    t.start()

    t.join()
```

### **3. –ê—Å–∏–Ω—Ö—Ä–æ–Ω–Ω–æ—Å—Ç—å (Asynchronous Programming)**  
**–ß—Ç–æ —ç—Ç–æ:**
